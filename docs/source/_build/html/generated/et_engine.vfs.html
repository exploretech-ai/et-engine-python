upload(local_file, remote_file, chunk_size=8388608, timeout=7200)[source]

Performs a multipart upload to s3

Steps: 1. Check the file’s size and determine the number of parts needed 2. Prepare the multipart upload with a POST request to Engine 3. Upload the parts with asynchronous POST requests to s3 with the presigned urls 4. Complete the multipart upload with another POST request to Engine with query string param ?complete=true
